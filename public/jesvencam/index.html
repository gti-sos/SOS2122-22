Descripción del Proyecto

Nuesto proyecto consta en analizar la relación existente entre las emisiones de CO2, la produccion de carbón y la balanza comercial entre diferentes paises en relacion a los ultimos años.

APIs realizadas en el curso 2021-2022

Mi api es la api co2-stats con datos organizados por pais y año donde se muestran además los datos de emisiones per cápita de CO2,emisiones de CO2 por cada 1.000 dólares de PIB y las emisiones de Co2 totales.
